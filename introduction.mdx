## What is Synto Mobile?

Synto Mobile is an **AI-powered mobile wallet interface** built for Solana that makes DeFi accessible to everyone. Instead of navigating complex menus and technical interfaces, simply chat with Synto in natural language to manage your crypto assets.

Built with React Native and Expo, Synto brings the power of decentralized finance to your mobile device with an intuitive, conversational interface powered by AI.

## Tech stack

Synto Mobile is built with cutting-edge technology:

- **Framework**: React Native with Expo for cross-platform mobile development
- **Navigation**: Expo Router with file-based routing
- **Blockchain**: Solana with Mobile Wallet Adapter integration
- **AI**: Vercel AI SDK for natural language chat interface
- **State Management**: TanStack Query for async state and caching
- **Storage**: AsyncStorage for local data persistence

<Note>
  Built for the **Solana Mobile Hackathon** by passionate builders committed to making DeFi accessible to everyone.
</Note>
